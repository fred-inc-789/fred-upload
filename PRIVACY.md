FRED Upload Extension — Privacy Policy
Effective Date: March 18, 2026 Published by: Fiduciary Real Estate Development, Inc. (FRED)
Overview FRED Upload is a Microsoft Edge browser extension developed for internal use by employees of Fiduciary Real Estate Development, Inc. It is not available to the general public.
Data Collected FRED Upload accesses the following information solely to perform its core function:
•	Microsoft account identity — used to authenticate the user via Microsoft Entra ID (Single Sign-On) and obtain an access token for OneDrive
•	Downloaded files — files downloaded in the browser are intercepted and uploaded to the user's company OneDrive account
•	File metadata — file names, types, and sizes are used to determine upload destination and user preferences
Data Use All data accessed by this extension is used exclusively to upload files to the authenticated user's Microsoft OneDrive account within the FRED Microsoft 365 tenant. No data is transmitted to any third-party service or stored outside of Microsoft's infrastructure.
Data Retention FRED Upload does not store personal data independently. File tokens are stored in browser session storage and are cleared when the browser session ends.
Third-Party Services This extension communicates exclusively with the Microsoft Graph API to perform OneDrive file operations. No other third-party services receive any data.
Contact For questions or concerns about this privacy policy, contact the FRED IT department at: fred.it@fred-inc.com
